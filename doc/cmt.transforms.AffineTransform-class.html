Performs an affine transformation on inputs.

$$\mathbf{\hat x} = \mathbf{P}_x(\mathbf{x} - \mathbf{m}_x),$$

Unlike AffinePreconditioner, this class does not touch the output and can thus be used with discrete data, for example.
