Class Hierarchy

  • object: The most base type
    • cmt.models.ConditionalDistribution: Abstract base class for conditional models.
      • cmt.models.GLM: An implementation of generalized linear models.
      • cmt.models.MCBM: An implementation of a mixture of conditional Boltzmann machines.
      • cmt.models.MCGSM: An implementation of a mixture of conditional Gaussian scale mixtures.
      • cmt.models.MLR: An implementation of multinomial logistic regression (also known as softmax regression).
      • cmt.models.STM: An implementation of the spike-triggered mixture model.
    • cmt.models.Distribution: Abstract base class for distributions.
      • cmt.models.Mixture: Basic mixture class implementing expectation maximization.
        • cmt.models.MoGSM: Mixture of Gaussian scale mixtures.
      • cmt.models.MixtureComponent: Abstract base class for objects which can be used as mixture components.
        • cmt.models.GSM: An implementation of a finite Gaussian scale mixture.
      • cmt.models.PatchModel: Abstract base class for models of image patches.
        • cmt.models.FVBN: Model image patches using a GLM for each conditional distribution.
        • cmt.models.PatchMCBM: Model image patches by using an MCBM for each conditional distribution.
        • cmt.models.PatchMCGSM: Model image patches by using an MCGSM for each conditional distribution.
      • cmt.models.UnivariateDistribution: Abstract base class for univariate distributions used, for example, by GLM.
        • cmt.models.Bernoulli: The Bernoulli distribution.
        • cmt.models.Binomial: The binomial distribution.
        • cmt.models.Poisson: The Poisson distribution.
    • cmt.nonlinear.Nonlinearity: Abstract base class for nonlinear functions used, for example, by GLM.
      • cmt.nonlinear.DifferentiableNonlinearity: Abstract base class for nonlinear functions used, for example, by GLM.
        • cmt.nonlinear.ExponentialFunction: The exponential function.
        • cmt.nonlinear.LogisticFunction: The sigmoidal logistic function.
      • cmt.nonlinear.InvertibleNonlinearity: Abstract base class for nonlinear functions used, for example, by GLM.
      • cmt.nonlinear.TrainableNonlinearity: Abstract base class for nonlinear functions used, for example, by GLM.
        • cmt.nonlinear.BlobNonlinearity: Mixture of Gaussian blobs.
        • cmt.nonlinear.HistogramNonlinearity: Histogram nonlinearity with $N$ bins.
        • cmt.nonlinear.TanhBlobNonlinearity: Hyperbolic tangent applied to Gaussian blob nonlinearity.
    • cmt.transforms.Preconditioner: Abstract base class for preconditioners of inputs and outputs.
      • cmt.transforms.AffinePreconditioner: Performs affine transformations on the input and the output.
        • cmt.transforms.AffineTransform: Performs an affine transformation on inputs.
          • cmt.transforms.BinningTransform: Sums up neighboring dimensions of input vectors.
          • cmt.transforms.PCATransform: This class behaves like PCAPreconditioner, but does not change the output.
          • cmt.transforms.WhiteningTransform: Decorrelates inputs.
        • cmt.transforms.PCAPreconditioner: This preconditioner can be used to reduce the dimensionality of the input.
        • cmt.transforms.WhiteningPreconditioner: Decorrelates inputs and outputs.
    • cmt.transforms.stackedaffinetransform.StackedAffineTransform: Concatenates affine transforms.
