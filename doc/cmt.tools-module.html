generate_data_from_image(img, input_mask, output_mask, num_samples=0)

 

Uniformly samples inputs and outputs for conditional models from images.

If no number of samples is specified, all possible inputs and outputs are extracted from the image and returned in row-major order.

Parameters:
  • img (ndarray) - an array representing a grayscale or color image
  • input_mask (ndarray) - a Boolean array describing the input pixels
  • output_mask (ndarray) - a Boolean array describing the output pixels
  • num_samples (int) - the number of generated input/output pairs
Returns: tuple
the input and output vectors stored in columns

extract_windows(time_series, window_length)

 

Extract windows from a given time series.

This method can be used to, for example, extract stimulus and spike windows before training a neuron model such as the STM.

>>> spikes = extract_windows(spike_train, 20)
>>> stimuli = extract_windows(stimulus, 20)

The last row of spikes contains the most recent bin of each window. We can use

>>> inputs = vstack([stimuli, spikes[:-1]])
>>> outputs = spikes[-1]

to generate the inputs and outputs to a neuron model.

>>> stm = STM(20, 50, 3, 10)
>>> stm.train(inputs, outputs)
Parameters:
  • time_series (ndarray) - an NxT array representing an N-dimensional time series of length T
  • window_length (int) - number of bins of the extracted windows
Returns: tuple
all possible overlapping windows of the time series

sample_spike_train(stimuli, model, spike_history=0, preconditioner=None)

 

Generate a spike train using a given model and a sequence of stimuli.

If the model's output depends on the spike history, the length of the spike history taken into account can be specified using spike_history. If spike_history is positive, the model at time $t$ will receive the stimulus at time $t$ concatenated with spike_history bins of the spike train preceding. The last entry of the input to the model is the most recent bin of the spike train.

If preconditioner is specified, the spike history is first transformed before appending it to the stimulus.

Parameters:
  • stimuli (ndarray) - each column represents a (possibly preprocessed) stimulus window
  • model (ConditionalDistribution) - a conditional distribution such as an STM
  • spike_history (int) - number of bins used as input to the model
  • preconditioner (Preconditioner) - transforms the spike history before appending it to the input
Returns: ndarray
sampled spike train

generate_masks(input_size, output_size=1, observed=None)

 

Generates some basic input and output masks.

If input_size is an integer, the number of columns of the mask will be that integer. If input_size is a list or tuple, a mask with multiple channels is created, which can be used with RGB images, for example.

By default, the input region will cover the upper half of the mask, also known as a *causal neighborhood*. If any of the channels is observed, the input region in that channel will cover a full square neighborhood around the output region.

Examples:

>>> input_mask, output_mask = generate_masks(8, 2)
>>> input_mask, output_mask = generate_masks([3, 7, 7], 1, [1, 0, 0])
Parameters:
  • input_size (int / list) - determines the size of the input region
  • output_size (int) - determines the size of the output region
  • observed (list) - can be used to indicate channels which are observed
Returns: tuple
one input mask and one output mask
