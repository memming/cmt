Basic mixture class implementing expectation maximization.

$$p(\mathbf{x}) = \sum_k \pi_k p_k(\mathbf{x})$$

To access the parameters $\pi_k$, use mixture.priors. Mixture components are added manually via mixture.add_component(component). To access component $k$, you can use mixture[k].

train(self, data, data_valid=None, parameters=None, component_parameters=None)

source code 

Fits the parameters of the mixture distribution to the given data using expectation maximization (EM).

The following example demonstrates possible parameters and default settings.

>>> model.train(data, data_valid, 
>>>     parameters={
>>>             'verbosity': 1,
>>>             'max_iter': 20,
>>>             'threshold': 1e-8,
>>>             'val_iter': 2,
>>>             'val_look_ahead': 5,
>>>             'initialize': True,
>>>             'train_priors': True,
>>>             'train_components': True,
>>>             'regularize_priors': 0.,
>>>     },
>>>     component_parameters={
>>>             'verbosity': 0,
>>>             'max_iter': 10,
>>>             'train_priors': True,
>>>             'train_mean': True,
>>>             'train_covariance': True,
>>>             'train_scales': True,
>>>             'regularize_priors': 0.,
>>>             'regularize_mean': 0.,
>>>             'regularize_covariance': 0.,
>>>             'regularize_scales': 0.,
>>>     },
>>> })
Parameters:
  • data (ndarray) - data points stored in columns
  • data_valid (ndarray) - validation data used for early stopping
  • parameters (dict) - hyperparameters controlling optimization and regularization
  • component_parameters (dict) - hyperparameters passed down to components during M-step
Returns: bool
True if training converged, otherwise False
