initialize(self, data, parameters=None)

source code 

Trains the model assuming shift-invariance of the patch statistics.

A single conditional distribution is fitted to the given data and all models with a complete neighborhood are initialized with this one set of parameters.

It is assumed that the patches are stored in row-order ('C') in the columns of data.

Parameters:
  • data (ndarray) - image patches stored column-wise

train(self, data, data_val=None, parameters=None)

source code 

Trains the model to the given image patches by fitting each conditional distribution in turn.

It is assumed that the patches are stored in row-order ('C') in the columns of data. If hyperparameters are given, they are passed on to each conditional distribution.

Parameters:
  • data (ndarray) - image patches stored column-wise
  • data_val (ndarray) - image patches used for early stopping based on validation error
  • parameters (dict) - a dictionary containing hyperparameters
Returns: bool
True if training of all models converged, otherwise False
